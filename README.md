# Li Lu · Investing Operating System

A Claude Code skill that distills Li Lu's investing framework into a runnable thinking advisor.

Li Lu is the founder of Himalaya Capital and the only outside money manager Charlie Munger ever entrusted with his family's capital.

## What's in this skill

- **5 mental models**: Compounding Ownership, Four-Quadrant Epistemology, Owner-Operator Lens, Dynamic Moat, Wealth as Relative Purchasing Power
- **9 decision heuristics**: buy/sell/sizing/too-hard/anti-shorting rules with real examples
- **Expression DNA**: his voice, vocabulary, certainty patterns, analogies, and style rules
- **Agentic Protocol**: researches real facts about a company before applying the framework — doesn't just opine from memory
- **Honest boundaries**: what the framework can't do, clearly stated
